# Gärtnerei Weeber – Moderne Website

![Next.js](https://img.shields.io/badge/Next.js-16-black?logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4-38BDF8?logo=tailwindcss)
![Vercel](https://img.shields.io/badge/Vercel-Deployed-black?logo=vercel)

Moderne, elegante Single-Page-Website für **Gärtnerei Weeber** – eine familiengeführte Gärtnerei in Eberdingen bei Stuttgart.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript** für Typsicherheit
- **Tailwind CSS v4** für Styling
- **Framer Motion** für Scroll-Animationen und Micro-Interactions
- **Google Fonts**: Cormorant Garamond (Display) + Plus Jakarta Sans (Body)

## Features

- Fullscreen Hero mit Parallax-Effekt und Blütenblatt-Animation
- Smooth Scroll Navigation
- Bento-Grid Service-Übersicht
- Animierte Counter
- Testimonial-Karussell
- Kontaktformular mit Validierung
- Google Maps Integration
- Responsive Design (Mobile-First)
- SEO-optimiert mit Schema.org JSON-LD
- Impressum & Datenschutzerklärung

## Lokale Entwicklung

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Deployment

Das Projekt wird automatisch über Vercel deployed. Jeder Push auf `main` löst ein neues Deployment aus.
